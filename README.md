
# 🚗 Automobile Sales Dashboard (Excel)

## 📌 Project Overview

An **interactive and dynamic Excel-based business intelligence dashboard** designed to analyze **multi-year automobile sales performance** across countries, cities, and product categories.

The dashboard provides actionable insights into **revenue, profit, taxation, and product-level performance** to support strategic decision-making and help stakeholders monitor business performance efficiently.

---

# 🎯 Short Description / Purpose

The **Automobile Sales Dashboard** is a data-driven Excel solution built to monitor and evaluate automobile sales performance from **2003–2005**.

It enables users to analyze:

* Revenue trends
* Profit margins
* Tax distribution
* Geographic sales performance
* Product-level profitability

through **interactive filters and visualizations**.

This dashboard is ideal for:

* Sales Managers
* Business Analysts
* Decision Makers

who need **quick insights into regional sales trends and product performance**.

---

# 🛠 Tech Stack

The dashboard was built using the following tools and technologies:

* 📊 **Microsoft Excel** – Core platform for dashboard development
* 📈 **Pivot Tables & Pivot Charts** – Dynamic aggregation and summarization
* 🎛 **Slicers** – Interactive filtering by Year, Country, City, and Product
* 🧮 **Advanced Excel Formulas** – KPI calculations such as Total Profit and Total Tax
* 🎨 **Dashboard Design & Formatting** – Custom layout, KPI cards, and consistent theme styling
* 📁 **File Format** – `.xlsx`

# 📂 Data Source

* Multiple countries

  * Austria
  * Belgium
  * Canada
  * Denmark
  * Finland
  * France
  * Germany
  * Ireland
  * and more

* Cities within each country

* Product categories including:

  * Classic Cars
  * Vintage Cars
  * Motorcycles
  * Trucks & Buses
  * Planes
  * Ships
  * Trains

The dataset includes:

* Sales Revenue
* Profit
* Tax values

This enables **hierarchical analysis**:

Country → City → Product → Year

and supports **multi-dimensional sales performance evaluation**.
